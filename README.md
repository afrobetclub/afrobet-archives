# AfroBet Club Archives

Production V1 for GitHub Pages.

## Files

- `index.html`
- `style.css`
- `config.js`
- `app.js`

## Deploy on GitHub

Upload the 4 files to the root of the `afrobet-archives` repository.

GitHub Pages URL will look like:

`https://afrobetclub.github.io/afrobet-archives/`

## Google Sheets

The app reads CSV from `config.js`.

If data does not load, publish the Google Sheet:

Google Sheets → File → Share → Publish to web → DATA tab → CSV

Then replace `CONFIG.CSV_URL` in `config.js`.
